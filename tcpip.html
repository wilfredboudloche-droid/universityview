TCP/IP Model Layers

Click on each layer to learn more about its role and protocols.

Link Layer
Handles physical network hardware and data framing. Protocols include Ethernet and Wi-Fi.
Internet Layer
Responsible for logical addressing and routing. Protocols include IP, ICMP.
Transport Layer
Manages end-to-end communication and error checking. Protocols include TCP and UDP.
Application Layer
Provides network services to applications. Protocols include HTTP, FTP, SMTP.
